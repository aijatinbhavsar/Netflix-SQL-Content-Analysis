# 🎬 Netflix Real-Time Content SQL Analysis
![Background Image](https://github.com/MadhurDwivedi/Netflix_Real_Time_Content_SQL_Analysis/blob/main/Background%20Image.jfif)
## 📌 Project Objective
The objective of this project is to analyze Netflix content data using SQL to understand content distribution, viewer trends, genre composition, country-wise production, and content characteristics.

This project focuses on evaluating movies and TV shows across multiple dimensions such as ratings, release years, countries, directors, actors, duration, and content descriptions. The analysis simulates a real-world OTT content analytics scenario, enabling insights into platform strategy, content planning, and audience preferences.

---

## 🗂️ Dataset Used
- The dataset is stored and managed within the database.
- It contains structured information related to:
  - Movies and TV shows
  - Directors and cast members
  - Countries of production
  - Release year and date added
  - Ratings and duration
  - Genres and descriptions
- The dataset is queried and filtered to derive meaningful analytical insights.
- 🔗 Dataset Link: <a href="https://github.com/MadhurDwivedi/Netflix_Real_Time_Content_SQL_Analysis/tree/main/Dataset">Dataset</a>

## ❓ Business Questions & Tasks (KPIs)
- Count the total number of Movies vs TV Shows on Netflix.
- Identify the most common ratings for movies and TV shows.
- List all movies released in a specific year (e.g., 2020).
- Find the top 5 countries with the highest number of Netflix content titles.
- Identify the longest-duration movie or TV show.
- Retrieve content added to Netflix in the last 5 years.
- List all movies and TV shows directed by Rajiv Chilaka.
- Identify TV shows with more than 5 seasons.
- Count the number of content items in each genre.
- Analyze yearly trends to find the top 5 years with highest average content released by India.
- List all content categorized as Documentaries.
- Identify content records without a director.
- Find how many movies actor Salman Khan appeared in during the last 10 years.
- Identify the top 10 actors appearing most frequently in Indian-produced movies.
- Categorize content as Good or Bad based on keywords like kill and violence in descriptions.

---

## ⚙️ Process
- Created a structured table to store Netflix content metadata.
- Performed exploratory analysis to understand content distribution.
- Executed analytical SQL queries using:
  - Aggregations
  - Filtering
  - Group By clauses
  - Conditional logic (CASE statements)
- Conducted time-based analysis using release year and date added.
- Performed text-based categorization using keyword matching.
- Generated insights to understand content trends and platform strategy.

## 🔍 Project Insights
- Netflix content is dominated by movies compared to TV shows.
- Certain countries contribute significantly more content than others.
- Content ratings vary clearly between movies and TV shows.
- Indian content shows strong growth trends in specific years.
- Genre distribution helps identify platform content focus areas.
- Keyword-based classification enables basic content sensitivity analysis.
- SQL is effective for large-scale OTT content analysis and reporting.

## ✅ Final Conclusion
This project demonstrates how SQL can be used to analyze OTT platform content data to uncover trends related to production, geography, genre, and content characteristics.
